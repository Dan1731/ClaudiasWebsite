## Motivation Behind Tool

After several years working in project controls and operations, I had demoed and implemented over fifteen project management platforms across different organizations. While many tools handled specific workflows well, none provided a clean balance between usability, reporting flexibility, and cost efficiency that worked for what I needed.

For the Professional Development Coursee, I decided to build the type of project management system I had always wanted to use.This project is currently being developed as a full-stack web application designed to manage projects, tasks, and reporting.

## Overview
The application is being developed using:
- Python with Flask  
- SQLite database for structured project storage  
- REST-style endpoints for future integrations  

- HTML/CSS for page layouts  
- Sidebar navigation for quick access to projects and reports  
- Expandable UI designed for future dashboards and analytics  

The data is focused around projects and tasks as separate primary entities.
The remaining features include: 
- Users  
- Clients  
- Status tracking  
- Budget and cost fields  

Each project stores:
- Timeline and projected completion dates  
- Budget and cost tracking  
- Percent complete  
- Workspace and client assignment  
- Status indicators

All reports will be custom to the user as they see fit among the features available to them for each section of the tool.
